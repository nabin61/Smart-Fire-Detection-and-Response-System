# Smart Forest Fire Detection System

## Overview
This project detects Smart Fire Detetction and Response system using sensors and ESP32.

## Technologies
- Flame Sensors
- L298 motor Driver
- Tip122 Transistor
- Resistors
- Servo Motors
- ESP32
- Arduino
- Aws (SNS, Lambda)


## Features
- Auto Fire Detection
- Auto Fire Suppression
- Real-time monitoring
- Email alerts
